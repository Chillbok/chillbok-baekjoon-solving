# 백준 문제 풀이(chillbok)

## Repository 소개

최근 나 자신의 코딩 실력이 매우 떨어진다는 것을 자각해 백준 문제를 풀어보고자 한다.
일단 목표는 일주일에 5문제, 가능하다면 하루에 한 문제 푸는 것이다.

### 현재 공부 중인 언어

- `C#`

### 공부 계획 중인 언어

- `Rust`
- `C++`
- `C`
- `Swift`
